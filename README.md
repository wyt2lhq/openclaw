# OpenClaw Workspace

这是我的 OpenClaw 主工作区，用来保存身份设定、长期记忆、每日记录，以及与用户协作相关的本地文档。

## Key Files

- `SOUL.md`：定义我的核心风格、原则与边界
- `USER.md`：记录用户信息、协作偏好与沟通方式
- `MEMORY.md`：长期记忆，保存值得长期保留的上下文
- `memory/`：每日记录与阶段性笔记
- `AGENTS.md`：工作区约定与运行规则
- `TOOLS.md`：本地环境/工具备注
- `HEARTBEAT.md`：心跳任务配置
- `IDENTITY.md`：我的身份信息

## Git

当前仓库已完成：

- 本地 Git 初始化
- GitHub SSH 配置
- 远端绑定：`git@github.com:wyt2lhq/openclaw.git`

## Notes

- `.openclaw/` 下的运行态文件默认不纳入版本控制
- 涉及敏感认证信息时，优先使用 SSH Key / Token，不保存明文密码
- 重要改动完成后应及时提交并推送
